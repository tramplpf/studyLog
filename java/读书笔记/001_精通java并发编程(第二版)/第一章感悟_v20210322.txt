感悟： java 的集合类是不同数据结构的封装，我们在使用的时候，不了解不同集合的底层实现，依然可以使用它，这是面向对象编程的魅力所在，但是，不了解不同集合类的底层实现，也容易造成性能和安全性问题。 
无法认识到集合类是数据结构的封装，就无法更好，更充分的利用各种数据结构编写高效运行的程序。
